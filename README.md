\# 📌 TP1 - Ingeniería de Software II



\## 📖 Descripción del proyecto



Este repositorio contiene el desarrollo del Trabajo Práctico 1 de la materia Ingeniería de Software II.



El objetivo principal es familiarizarse con el uso de Git, GitHub y la gestión de proyectos.



\---



\## 🎯 Objetivos



\### Objetivo general



\- Comprender el uso de control de versiones con Git



\### Objetivos específicos



1\. Crear un repositorio en GitHub  

2\. Gestionar archivos y directorios  

3\. Versionar cambios en el código  

4\. Recuperar archivos eliminados  



\---



\## 📂 Estructura del proyecto



\- src → código fuente  

\- doc → documentación  

\- bin → ejecutables  

\- script → scripts auxiliares  



\---



\## ⚙️ Funcionalidad principal



El archivo `primes.py` permite:



1\. Generar números primos  

2\. Mostrar resultados en pantalla  



\---



\## 🧪 Pasos realizados



1\. Instalación de herramientas (Python y Git)  

2\. Creación del repositorio  

3\. Subida de archivos  

4\. Simulación de borrado  

5\. Recuperación con Git
